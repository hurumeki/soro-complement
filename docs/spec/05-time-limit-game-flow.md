# 5. Time Limit & Game Flow

```
[Title Screen]
  → Difficulty selection
  → [Game Screen] 3-second countdown then start
  → 1-minute play session
  → [Time-up animation]
  → [Result Screen] Score & high score display
  → [Return to title / Play again]
```

## Timer Specification

- Remaining time displayed as a progress bar + numeric value
- Warning animation at 10 seconds remaining (color change, flashing)
- Abacus controls lock immediately at time-up
