# 2. Screen Layout

```
┌────────────────────────────────┐
│  [Time Left] [Score]           │  ← Header bar
│                                │
│    ╔══╦══╦══╗                  │
│    ║  1 ║  2 ║  3 ║            │  ← Problem area (at the tip of abacus rods)
│    ╚══╩══╩══╝                  │
│                                │
│  ┌──┬──┬──┐                   │
│  │  │  │  │  ← Upper bead area │
│  ├──┼──┼──┤                   │
│  │  │  │  │  ← Beam (hari)     │  ← Abacus area
│  ├──┼──┼──┤                   │
│  │  │  │  │                   │
│  │  │  │  │  ← Lower bead area │
│  │  │  │  │                   │
│  └──┴──┴──┘                   │
│                                │
│  [Current: 000]  [Clear]       │  ← Footer bar
└────────────────────────────────┘
```

- **Portrait orientation** recommended
- The problem area and abacus area are aligned vertically, so each digit visually corresponds to an abacus rod
- Number of rods (digits) ranges from 3 to 5 (depends on difficulty; see section 4)
