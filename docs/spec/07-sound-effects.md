# 7. Sound Effects (Web Audio API)

For offline compatibility, no external audio files are used. All sounds are generated programmatically via the Web Audio API.

| Event                | Sound Type |
|----------------------|------------|
| Bead movement        | Short click (wood collision sound) |
| Carry-over           | Rising tone (short beep) |
| Clear                | Achievement chime (bright) |
| Combo clear          | More elaborate chime |
| Time-up              | End sound |
| Operation blocked    | Low buzzer |
